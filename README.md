# 📝 To-Do List App

A simple and clean **To-Do List web application** built using **HTML, CSS, and JavaScript**.
This app allows users to add tasks, mark them as completed, delete tasks, and automatically save them in the browser using **localStorage** so they remain even after refreshing the page.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Tasks are saved using **localStorage**
* 🔄 Tasks remain after page refresh
* 🎨 Simple and responsive UI

---

## 🛠️ Technologies Used

* **HTML** – Structure of the app
* **CSS** – Styling and layout
* **JavaScript** – Functionality and task management
* **localStorage** – Persisting tasks in the browser

## ⚙️ How It Works

1. The user enters a task in the input box.
2. Clicking the **Add button** creates a new `<li>` task element.
3. A **delete button (×)** is added to each task.
4. Clicking a task toggles the **completed state**.
5. Tasks are stored using **localStorage**.
6. When the page reloads, saved tasks are loaded back automatically.
